## Basico de git
* gid add "archivo a agregar"
* git add .  (agrega todo lo distinto cuidado de no agregar cosas que no pinten)
* git commit -m "texto del commit"
* git push  (sube al repo)
* git pull  (baja cambios en el repo)
* git clone "link_repo" (clona repo en la pc)
* Sino es posible subir directo en la pagina
